    Lenguaje Php


    El PHP es un lenguaje de scripting de código abierto, del lado del servidor, con programación HTML integrada que se utiliza para crear páginas web dinámicas. Las ventajas de PHP son su flexibilidad y su alta compatibilidad con otras bases de datos. Además, PHP es considerado como un lenguaje fácil de aprender.

    PHP se relacionan con los scripts del lado del servidor. PHP puede realizar cualquier tarea que cualquier programa CGI o Common Gateway Interface puede hacer y maneja el intercambio de datos entre el servidor y el software. Por lo tanto, PHP puede recopilar datos o crear webs de contenido dinámico.

    ¿A que tipo de paradigma pertenece?

    - imperativo, funcional, orientado a objetos, procedural, reflexivo


    Con el lenguaje PHP es posible emplear el paradigma orientado a objetos para programar, como así también el estructurado e incluso combinarlos. Ademas que también sigue los paradigmas de programación imperativa y la programación reflexiva, esto porque php es un lenguaje tan variado y extenso que su metodologia multiparadigma abarca distintos paradigmas.
