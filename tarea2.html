    • La clasificacion de los lenguajes informaticos


    Los lenguajes informáticos se pueden clasificar de esta forma:

    • Lenguaje de programación
    • Lenguaje de especificación
    • Lenguaje de consulta.
    • Lenguaje de marcas.
    • Lenguaje de transformación.
    • Protocolo de comunicaciones.
    • Lenguaje de sonido.
    • Lenguaje gráfico.
    • Pseudocódigo
    • Lenguaje de Definición de Procesos.



    Lenguaje de programación


    Un lenguaje de programación es un lenguaje de computadora que los programadores utilizan para comunicarse y para desarrollar programas de software, aplicaciones, páginas webs, scripts u otros conjuntos de instrucciones para que sean ejecutadas por los ordenadores, asi como se utilizana para el propio desarrollo de otras aplicaciones, ya sean para diferentes puertos o plataformas tales como mobil, pc, laptop, ipad etc. hay muchos tipos de programas que se pueden hacer gracias a los lenguajes de programacion.

    Pero en si, en la informatica al lenguaje de prorgamacion se le conoce como un programa destinado a la construccion de otros programas informaticos, esa es la forma mas resumida y completa de la principal funcion del lenguaje de programacion.



    Ejemplo: (Java y C++)





    Lenguaje de especificación


    El lenguaje de especificación o lenguaje de descripción es un lenguaje formal osemi-formal cuya principal función es construir modelos de los sistemas que se desea elaborar. Diferencia entre lenguaje de especificación y lenguaje de programación. Además, que, en el contexto de la ingeniería eléctrica, la computación y ramas afines, un lenguaje de es pecificación o lenguaje de descripción es un lenguaje formal o semi-formal cuya función es construir modelos de los sistemas que se desea elaborar.

    Pero en si, el lenguaje de especificacion como s emenciono anteriormente permite el construir modelos de los sistemas que se desean editar o elaborar, un ejemplo de este tipos de lenguaje podriamos mencionar el lenguaje CSS, ya que este te permite poder modificar caracteristicas y etiquetas especificas de la pagina web html.



    Ejemplo: (Css)





    Lenguaje de consulta.


    Principalmente un lenguaje de consulta es un lenguaje informático que es usado para hacer distintas consultas en bases de datos y sistemas de información. Además, que los lenguajes de consulta pueden ser clasificados de acuerdo si son lenguajes de consulta de bases de datos o lenguajes de consulta de recuperación de información, asi haciendo la consulta de la base de deatos algo funfamental para cualquier aplicacion, pagina o aplicacion.

    Ademas que este tipo de lenguaje permite gracias a la peticion de la base de daosm permite guardar distintos valores para asi crear distintos



    Ejemplo: (SQL)





    Lenguaje de marcas.


    Un lenguaje de marcas principalmente es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación. Además, que los lenguajes de marcado suelen confundirse con lenguajes de programación. Pero, sin embargo, estos no son lo mismo, ya que el lenguaje de marcado no tiene funciones aritméticas o variables, como poseen los lenguajes de programación.



    Ejemplo: (Html y XML)





    Lenguaje de transformación.


    Para el lenguaje de transformación explicaremos mejor este tema tomando como ejemplo el lenguaje o programa que mejor lo representa que seria el XSLT, este es un lenguaje que permite aplicar una transformación a un documento XML para obtener otro documento XML, un documento HTML o un documento de texto plano, así en pocas palabras sería la mejor descripción para un lenguaje de transformación.



    Ejemplo: (XSLT)





    Protocolo de comunicaciones.


    Principalmente en la informática y telecomunicación, un protocolo de comunicaciones es un sistema de reglas que permiten que dos o más entidades computadoras, teléfonos celulares, de un sistema de comunicación se comuniquen entre ellas para transmitir información por medio de cualquier tipo de variación de una magnitud física.

    Además se trata de las reglas o el estándar que define la sintaxis, semántica y sincronización de la comunicación, así como también los posibles métodos de recuperación de errores. Los protocolos pueden ser implementados por hardware, por software, o por una combinación de ambos.
