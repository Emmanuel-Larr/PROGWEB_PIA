    Lenguajes de programacion


    Un lenguaje de programacion es con el que mediante una serie de instrucciones, le permite a un programador escribir un conjunto de órdenes, acciones consecutivas, datos y algoritmos para, de esa forma, crear programas que controlen el comportamiento físico y lógico de una máquina.

    Tambien se podria decir que un lenguaje de programación es un lenguaje de computadora que los programadores utilizan para comunicarse y para desarrollar programas de software, aplicaciones, páginas webs, scripts u otros conjuntos de instrucciones para que sean ejecutadas por los ordenadores, asi como se utilizana para el propio desarrollo de otras aplicaciones, ya sean para diferentes puertos o plataformas tales como mobil, pc, laptop, ipad etc.



    Clasificación de los lenguajes de alto nivel


    En si, esta seria la clasificacion principal de los lenguajes de alto

    • Lenguajes imperativos o procedurales
    • Lenguajes imperativos o procedurales
    • Lenguajes orientados a objetos
    • Lenguajes orientados al problema



    1.- Lenguajes imperativos o procedurales


    Estos lenguajes principlemnte se fundamentan en el uso de variables para almacenar valores y el uso de instrucciones que indican las operaciones a realizar sobre los datos. Son realmente eficientes ya que no son muy dificiles de usar y los que estan disponibles los puedes descargar gratis. La mayoría de los lenguajes de alto nivel son de este tipo.



    2.- Lenguajes imperativos o procedurales


    Para este lenguaje en este caso, el proceso por el cual se ejecuta el programa no aparece de forma explícita, ya que los programas se construyen mediante la definición de funciones que pueden ser lenguajes funcionales como Lisp o expresiones lógicas que indican las relaciones entre determinadas estructuras de datos o lenguajes de programación lógica como Prolog.



    3.- Lenguajes orientados a objetos


    En este lenguaje el diseño de los programas se centra más en los datos y su estructura, esto ya que los programas consisten en descripciones de unidades denominadas objetos que encapsulan los datos y las operaciones que actúan sobre ellos. Uno de los lenguajes más usados en esta filosofía es el C++.



    4.- Lenguajes orientados al problema


    Este tipo de lenguajes están diseñados para problemas específicos, principalmente de gestión, ya que los programas están formados por sentencias que indican qué se quiere hacer. Generalmente, suelen ser generadores de aplicaciones que permiten cierta automatización de la tarea de desarrollo de software de gestión.



    Lenguajes interpretados


    Los lenguajes interpretados se le conoce como un lenguaje cuyo código no necesita ser preprocesado mediante un compilador, eso significa que el ordenador es capaz de ejecutar la sucesión de instrucciones dadas por el programador sin necesidad de leer y traducir exhaustivamente todo el código. Para que esto sea posible y pueda funcionar hace falta un intermediario, que es un programa encargado de traducir cada instruccion escrita con una semantica 'humana' a Código máquina tambien seria como las instrucciones de la CPU del ordenador, este programa recibe el nombre de interprete en el idioma ingles parser.

    Pero en si, un lenguaje de interpretacion es un lenguaje de programación en el que la mayoría de sus implementaciones ejecutan las instrucciones directamente, sin una previa compilación del programa a instrucciones en lenguaje máquina. Ademas que los lenguajes interpretados también pueden diferenciarse de los lenguajes de máquina.

    Lenguajes compilados


    Un lenguaje compilado es un lenguaje de programación de computadora cuyo código fuente generalmente se compila o traduce al código de máquina para producir un programa ejecutable, por lo general los compiladores son programas de utilidad creados para traducir una implementación específica de un lenguaje de programación en un archivo binario ejecutable que está diseñado para ejecutarse en un determinado sistema operativo. Los lenguajes compilados tambien pueden usarse para crear programas que controlen el comportamiento físico y lógico de una máquina, estopara expresar algoritmos con precisión, o como modo de comunicación.

    Ademas que los propios lenguajes compilados están formados por un conjunto de símbolos y reglas sintácticas y semánticas que definen su estructura y el significado de sus elementos y expresiones. Al proceso por el cual se escribe, se prueba, se depura, se compila de ser necesario y se mantiene el código fuente de un programa informático se le llama programación.

    Lenguajes orientados a objetos


    La programación orientada a objetos o Object Oriented Programming, tambien abreviado OOP es un modelo de programación informática que organiza el diseño de software en torno a datos u objetos, en lugar de funciones y lógica. Un objeto se puede definir como un campo de datos que tiene atributos y comportamiento únicos.

    Ademas que lo mismo que la programación estructurada, los conceptos de POO se pueden implementar usando lenguajes tales como C y PASCAL, sin embargo, la programación se hace pesada y puede generar confusión cuando los programas crecen de tamaño. Un lenguaje que este especialmente diseñado para soportar los conceptos de POO se vuelve mas adecuado para implementarlos.

    Lenguajes funcionales


    Un lenguaje de programación funcional principalmente es un lenguaje de programación declarativo donde el programador especifica lo que quiere hacer, en lugar de lidiar con el estado de los objetos. Es decir, las funciones estarían en un primer lugar y nos centraremos en expresiones que pueden ser asignadas a cualquier variable.

    Ademas que en un lenguaje de programacion funcional, todos los elementos pueden entenderse como funciones y el código puede ejecutarse mediante llamadas de función secuenciales. Por el contrario, no se asignan valores de forma independiente. Una función se imagina mejor como una variante especial de un subprograma. Esta es reutilizable y, a diferencia de un procedimiento, devuelve directamente un resultado.
